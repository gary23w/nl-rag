# Vagrant Documentation

Vagrant is the command line utility for managing the lifecycle of virtual machines.

- CLI
- Vagrant Cloud
- Vagrantfile

## Fundamentals

- BoxesThe package format for Vagrant environments - create your own or use one from the public catalog.
- VagrantfileDescribe your virtual machine's configuration and how to provision it.
- ProvisioningProvisioners allow you to automatically install software, alter configurations, and more.
- The 'up' commandCreate and configure your virtual machine with the single most important Vagrant CLI command.
- Synced FoldersSync files from your host machine to the guest machine.
- NetworkingAccess your virtual machines through the high-level networking set up by Vagrant.

## HCP Vagrant Registry

HCP Vagrant Registry serves a public, searchable index of Vagrant boxes.

- CatalogStore and version your Vagrant boxes in HCP Vagrant Registry.
- Configure permissionsConfigure HCP Vagrant Registry permissions.
- Reclaim registriesReclaim registries from Vagrant Cloud.
