# **Pulumi Docs**

Pulumi is an open source platform for building, deploying, and managing cloud infrastructure using your favorite programming languages. Manage infrastructure, secrets, and configuration with a unified workflow across any cloud.

Get Started

## Capabilities

#### Infrastructure as Code

Define and manage cloud infrastructure using TypeScript, Python, Go, .NET, Java, and YAML.

#### Deployments & Workflows

Cloud-hosted deployments, drift detection, state management, and automation.

#### Secrets & Configuration

Centralized secrets and configuration management with environments.

#### Insights & Governance

Search, compliance, and policy enforcement across your cloud infrastructure.

#### Version Control

Connect Pulumi with GitHub, GitLab, and Azure DevOps using Pulumi-maintained version control integrations.

#### Internal Developer Platform

Self-service infrastructure with templates, guardrails, and developer portals.

#### Infrastructure AI

Infrastructure automation with Pulumi Neo and natural language assistance.
