# Materialize documentation

Materialize is the live data layer for apps and AI agents. To keep results up-to-date as new data arrives, Materialize incrementally updates results as it ingests data rather than recalculating results from scratch.

## What’s new!

- **MCP servers and Agent skills**:
  - MCP server for agents: give your production AI agents fresh context from Materialize
  - MCP server for developers: give coding agents observability into your Materialize environment
  - Agent skills

For more information on these and other changes, see the Release Notes.
