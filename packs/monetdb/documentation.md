# Welcome to the MonetDB Documentation

##### Getting started, SQL manual, Tutorials

This guide aims at people primarily interested into using MonetDB directly to develop SQL-based applications using its command-line-interface or developing JDBC & ODBC applications.

##### Monitoring, Features, Man pages

This section provides an in-depth documentation for server administrators. There is information about installation (from source), updates, security and roles, backup and monitoring.

##### Components, SQL extensions, Debugging

This guide aims at database application developers as well as those that seek to improve the MonetDB code base itself.
