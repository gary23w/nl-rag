# User Guide

With its origins in the database research community, MonetDB continues to break new ground as a database management system applicable to real-world use cases. Its compatibility with a wide range of platforms renders it easy to set up, use and administer.

MonetDB comprises an elaborate ecosystem capable of addressing complex data management challenges. The topics listed below allow new users to gain a foothold in the MonetDB suite. Particular attention has been placed on the SQL syntax and features in order to bolster the process of getting an application up and running.

- Getting Started Installation guide and tutorial
- Introduction to MonetDB Key features, concepts and historic perspectives
- Client Interfaces Start programming database applications
- Loading Data into MonetDB Bulk data ingestion via external files (e.g. CSV format, for example)
- SQL Manual Basic knowledge of SQL is expected in order effectively utilise this synopsis of the SQL 2008 language features supported by MonetDB and extensions to the SQL standard.
- Blog Archive Collection of blog posts about MonetDB
