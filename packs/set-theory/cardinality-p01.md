# Cardinality

In mathematics, **cardinality** is an inherent property of sets, roughly meaning the number of individual objects they contain, which may be infinite. The concept is understood through one-to-one correspondences between sets. That is, if their objects can be paired such that each object has a pair, and no object is paired more than once.

Two sets are said to be ***equinumerous*** or *have the same cardinality* if there exists a one-to-one correspondence between them. Otherwise, under the axiom of choice, one of the two sets must be equinumerous with a strict subset of the other and is said to be *strictly smaller* than it; the other set is *strictly larger*. Using this concept, it is possible to show there are different sizes of infinity.

A set is countably infinite if it can be placed in one-to-one correspondence with the set of natural numbers Ôüá $\{1,2,3,4,\cdots \}$ Ôüá. For example, the set of even numbers Ôüá $\{2,4,6,\cdots \}$ Ôüá and the set of rational numbers are countable. Uncountable sets are those strictly larger than the set of natural numbers. The set of all real numbers and the powerset of the set of natural numbers are proven to be uncountable by so-called diagonal arguments. Cantor's theorem generalizes these arguments to show there is an infinite hierarchy of infinities.

For finite sets, cardinality recovers the usual concept of size as "number of elements." However, it is more often difficult to ascribe "sizes" to infinite sets. A system of cardinal numbers can be developed to extend the role of natural numbers in answering "how many". Most commonly, the Aleph numbers Ôüá $\aleph _{0},\aleph _{1},\aleph _{2},...\aleph _{\omega },\aleph _{\omega +1}...$ Ôüá are used, since their definition naturally extends the process of counting, and it can be shown that every infinite set has cardinality equivalent to some Aleph.

The set of natural numbers has cardinality Ôüá $\aleph _{0}$ Ôüá. The question of whether the real numbers have cardinality Ôüá $\aleph _{1}$ Ôüá is known as the continuum hypothesis, which has been shown to be both unprovable and undisprovable in standard set theories such as ZermeloÔÇôFraenkel set theory. Alternative set theories and additional axioms give rise to different properties and have often strange or unintuitive consequences. However, every theory of cardinality using standard logical foundations of mathematics admits Skolem's paradox.

The basic concepts of cardinality go back as early as the 6th century BCE, and there are several close encounters with it throughout history, however, the results were generally dismissed as paradoxical. It is considered to have been first introduced formally to mathematics by Georg Cantor at the turn of the 20th century. Cantor's theory of cardinality was then formalized, popularized, and explored by many influential mathematicians of the time, and has since become a fundamental concept of mathematics.


## Basics

### Definition

Cardinality is an inherent property of sets which defines their size, roughly meaning the number of individual objects they contain. Fundamentally however, it is different from the concepts of number or counting as the cardinalities of two sets can be compared without referring to their number of elements, or defining number at all. For example, in the image above, a set of apples is compared to a set of oranges such that every fruit is used exactly once which shows these two sets have the same cardinality, even if one does not know how many of each there are. Thus, cardinality is measured by putting sets in so-called one-to-one correspondence: each object in one set can be assigned a dedicated "pair" in the other, and no object from either set left unpaired. If it is possible, the sets are said to have the *same cardinality*, and if not, one set is said to be *strictly larger* or *strictly smaller* than the other.

The basic concepts of cardinality are developed in terms of sets and functions, which are more abstract than their counterparts outside of mathematics. Informally, a set can be understood as any collection of objects, usually represented with curly braces. For example, Ôüá $S=\{1,2,3\}$ Ôüá specifies a set, called Ôüá S Ôüá, which contains the numbers 1, 2, and 3. The symbol Ôüá $\in$ Ôüá represents set membership, for example Ôüá $1\in S$ Ôüá says "1 is a member of the set Ôüá S Ôüá" which is true by the definition of Ôüá S Ôüá above. Here Ôüá S Ôüá is finite, but that is not a requirement in general. The only requirement for a set is that it is well-defined. That is, for any object, Ôüá x Ôüá, one can determine whether Ôüá x Ôüá belongs to that set Ôüá $(x\in S)$ Ôüá, or Ôüá x Ôüá does not belong to that set Ôüá $(x\notin S)$ Ôüá. One example of an infinite set is the set of all natural numbers Ôüá $\{1,2,3,\cdots \}$ Ôüá.

A function, or correspondence, maps each member of one set to a unique member of another, often represented with an arrow diagram. For example, the adjacent table depicts several functions which map sets of natural numbers to sets of letters. If a function does not map two members to the same place, it is called injective. If a function covers every member in the output set, it is called surjective. If a function is both injective and surjective, it is called bijective or a one-to-one correspondence. Functions are not limited to those one can draw an arrow diagram for, so long as the function is well-defined. That is, for each possible input, the output is defined. For example, one may define a function Ôüá f Ôüá on the natural numbers by multiplying by two: Ôüá $f(1)=2,\;$ ÔüáÔüá $f(2)=4,\;$ ÔüáÔüá $f(3)=6,\dots \,$ Ôüá Ôüá $f(n)=2n\dots$ Ôüá

The term *cardinality* originates from the post-classical Latin *cardo* ("to hinge"), which referred to something central or pivotal, both literally and metaphorically. This passed into medieval Latin and then into English, where *cardinal* came to describe things considered to be, in some sense, fundamental, such as, *cardinal sins*, *cardinal directions*, and (in linguistics) *cardinal numbers*. The last of which referred to numbers used for counting (e.g., *one*, *two*, *three*), as opposed to *ordinal numbers*, which express order (e.g., *first, second, third*), and *nominal numbers* used for labeling without meaning (e.g., jersey numbers and serial numbers).

In mathematics, the notion of cardinality was first introduced by German mathematician Georg Cantor in the late 19th century, who used the term *M├ñchtigkeit*, which may be translated as "magnitude" or "power", though Cantor credited the term to a work by Jakob Steiner on projective geometry. Around 1930, the terms *cardinality* and *cardinal number* were adopted from the grammatical sense, and later translations would use these terms.


## Comparing sets

### Equinumerosity

An intuitive relationship between two sets having the "same size" is that their objects can be paired one-to-one. That is, if each object in one set can be assigned a dedicated "pair" in the other, and no object from either set left unpaired, there are the same number of objects in each set. A one-to-one pairing between two sets defines a bijective function between them by mapping each object to its pair. Similarly, a bijection between two sets defines a pairing of their elements by pairing each object with the one it maps to. Therefore, these notions of "pairing" and "bijection" are, at least intuitively, equivalent. Thus, the following definition is given:

Two sets are said to have the *same cardinality* or be *equinumerous* if their members can be paired one-to-one. That is, if there exists a function between them which is bijective. This is often written as Ôüá $A\sim B$ Ôüá, or Ôüá $A\approx B$ Ôüá. Alternatively, these sets may be said to be *equivalent*, *similar*, *equipotent*, or *equipollent*. For example, the set Ôüá $E=\{0,2,4,6,{\text{...}}\}$ Ôüá of even numbers has the same cardinality as the set Ôüá $\mathbb {N} =\{0,1,2,3,{\text{...}}\}$ Ôüá of natural numbers, since the function Ôüá $f(n)=2n$ Ôüá is a bijection from Ôüá $\mathbb {N}$ Ôüá to Ôüá E Ôüá.

The property for finite sets that "the whole is greater than the part" is no longer true for infinite sets, and the existence of surjections or injections that do not work does not prove that there is no bijection. For example, the function Ôüá g Ôüá from Ôüá $\mathbb {N}$ Ôüá to Ôüá E Ôüá, defined by multiplying by 4, Ôüá $g(n)=4n$ Ôüá, is injective, but not surjective (since 2, for instance, is not mapped to). Further, the function Ôüá h Ôüá from Ôüá $\mathbb {N}$ Ôüá to Ôüá E Ôüá, defined by rounding down to the nearest even number, Ôüá $h(n)=2\operatorname {floor} (n/2)$ Ôüá (cf. *floor function*), is surjective, but not injective, (since 0 and 1 for instance both map to 0). Neither Ôüá g Ôüá nor Ôüá h Ôüá can challenge Ôüá $E\sim \mathbb {N}$ Ôüá, which was established by the existence of Ôüá f Ôüá.

#### Equivalence

A fundamental result in developing a theory of cardinality is that equinumerosity forms an equivalence relationÔÇöthat is, a relation satisfying the same three basic properties as equality: reflexivity, symmetry, and transitivity.

Reflexivity, the property that every set has the same cardinality as itself Ôüá $(A\sim A)$ Ôüá, follows from the identity function: for any set Ôüá A Ôüá, the function that maps each element to itself is a bijection from Ôüá A Ôüá to **Ôüá A Ôüá**. Symmetry, the property that, if Ôüá A Ôüá has the same cardinality as Ôüá B Ôüá Ôüá $(A\sim B)$ Ôüá, then Ôüá B Ôüá has the same cardinality as Ôüá A Ôüá Ôüá $(B\sim A)$ Ôüá, holds because any bijection Ôüá $f:A\to B$ Ôüá has an inverse function Ôüá $f^{-1}:B\to A$ Ôüá, which is also bijective. Transitivity, the property that if Ôüá A Ôüá and Ôüá B Ôüá have the same cardinality Ôüá $(A\sim B)$ Ôüá, and Ôüá B Ôüá and Ôüá C Ôüá have the same cardinality Ôüá $(B\sim C)$ Ôüá then so do Ôüá A Ôüá and Ôüá C Ôüá Ôüá $(A\sim C)$ Ôüá, follows from the composition of functions: given bijections Ôüá $f:A\to B$ Ôüá and Ôüá $g:B\to C$ Ôüá, their composition Ôüá $g\circ f$ Ôüá is a bijection from Ôüá A Ôüá to Ôüá C Ôüá (see image).

Since equinumerosity satisfies all three properties, it is an equivalence relation. This means that it groups sets into equivalence classesÔÇögroups of sets that are all equinumerous with one anotherÔÇöwhere each group defines a possible size of a set. This motivates the notion of cardinal numbers, which are representatives that stand for the "size" of each group, developed in the section below.

### Inequality

A set Ôüá A Ôüá is not larger than a set Ôüá B Ôüá if it can be mapped into Ôüá B Ôüá without overlap. That is, the cardinality of Ôüá A Ôüá is less than or equal to the cardinality of Ôüá B Ôüá if there is an injective function from Ôüá A Ôüá to **Ôüá B Ôüá**, or equivalently, if there is a surjection from **Ôüá B Ôüá** to Ôüá A Ôüá (or Ôüá A Ôüá is empty). This is written Ôüá $A\preceq B$ Ôüá, or Ôüá $A\lesssim B$ Ôüá. and read as "Ôüá A Ôüá is *not greater than* Ôüá B Ôüá," or "Ôüá A Ôüá is *dominated by* Ôüá B Ôüá." If Ôüá $A\preceq B$ Ôüá, but there is no injection from Ôüá B Ôüá to Ôüá A Ôüá, then Ôüá A Ôüá is said to be *strictly* smaller than Ôüá B Ôüá, written without the underline as Ôüá $A\prec B$ Ôüá. For example, if Ôüá A Ôüá has four elements and Ôüá B Ôüá has five, then the following are true Ôüá $A\preceq A$ Ôüá, Ôüá $A\preceq B$ Ôüá, and Ôüá $A\prec B$ Ôüá.

The basic properties of an inequality are reflexivity (for any Ôüá a Ôüá, Ôüá $a\leq a$ Ôüá), transitivity (if Ôüá $a\leq b$ Ôüá and Ôüá $b\leq c$ Ôüá, then Ôüá $a\leq c$ Ôüá) and antisymmetry (if Ôüá $a\leq b$ Ôüá and Ôüá $b\leq a$ Ôüá, then Ôüá $a=b$ Ôüá). Cardinal inequality Ôüá $(\preceq )$ Ôüá as defined above is reflexive since the identity function is injective, and is transitive by function composition. AntisymmetryÔÇöthe fact that, given injections in both directions, then one can find a bijectionÔÇöis non-trivial, and is the content of the so-called Schr├ÂderÔÇôBernstein theorem. One such proof can be summarized as follows, with the help of the adjacent illustration:

Given injections in both directions (Ôüá $f:A\to B$ Ôüá, and Ôüá $g:B\to A$ Ôüá), take set Ôüá A Ôüá and "color" each point blue. Then take the image of set Ôüá B Ôüá on Ôüá A Ôüá after the injectionÔÇöthat is, the set of points that the injection projects on toÔÇöand color those red. Then take the image of Ôüá A Ôüá on this image of Ôüá B Ôüá, and color those points blue again. By recursively alternating the images of these sets, one obtains a "coloring" of the original set (see illustration). For each point, it either stops at some finite stepÔÇöand thus is definitively colored blue or redÔÇöor continues switching indefinitely. Of the points in Ôüá A Ôüá that are definitively colored blue, map them into the next recursive image (i.e. by applying Ôüá f Ôüá then Ôüá g Ôüá), leaving all other points in place. This transforms the set Ôüá A Ôüá into exactly the image of Ôüá B Ôüá. Taking this transformation, followed by the inverse of Ôüá g Ôüá, gives a bijection from Ôüá A Ôüá to Ôüá B Ôüá (since Ôüá g Ôüá is injective and surjective on its image, its inverse is bijective).

A further property of cardinal inequality is totality, which says that any two sets are comparable. That is, for any Ôüá a Ôüá and Ôüá b Ôüá, either Ôüá $a\leq b$ Ôüá or Ôüá $b\leq a$ Ôüá. A full proof of which requires concepts introduced later, however the argument can be briefly summarized as follows. Every well-ordered set is isomorphic to a unique ordinal number, called the order type of the set. By the well-ordering theorem, every set can be well-ordered. Then, by comparing their order types, one can show that Ôüá $A\preceq B$ Ôüá or Ôüá $B\preceq A$ Ôüá. This fact is equivalent to the axiom of choice.


## Countability

### Countable sets

A set is called *countable* if it is finite or has a bijection with the set of natural numbers Ôüá $(\mathbb {N} )$ Ôüá, in which case it is called *countably infinite*. The term *denumerable* is also sometimes used for countably infinite sets. For example, the set of all even natural numbers is countable, and therefore has the same cardinality as the whole set of natural numbers, even though it is a proper subset. Similarly, the set of square numbers is countable, which was considered paradoxical for hundreds of years before modern set theory (cf. *┬º Pre-Cantorian set theory*). However, several other examples have historically been considered surprising or initially unintuitive since the rise of set theory.

The rational numbers Ôüá $(\mathbb {Q} )$ Ôüá are those which can be expressed as the quotient or fraction Ôüá ${\tfrac {p}{q}}$ Ôüá of two integers. The rational numbers can be shown to be countable by considering the set of fractions as the set of all ordered pairs of integers, which can be visualized as the set of all integer points on a grid. Then, an intuitive function can be described by drawing a line in a repeating pattern, or spiral, which eventually goes through each point in the grid. For example, going through each diagonal on the grid for positive fractions, or through a lattice spiral for all integer pairs. These over-cover the rationals, since, for example, the fractions Ôüá $\textstyle {\frac {2}{4}},\,{\frac {3}{6}},\,{\frac {4}{8}},\,\dots$ Ôüá all get mapped to Ôüá $\textstyle {\frac {1}{2}}$ Ôüá, as the grid method treats these all as distinct ordered pairs. So this function shows Ôüá $\vert \mathbb {Q} \vert \leq \vert \mathbb {N} \vert$ Ôüá not Ôüá $\vert \mathbb {Q} \vert =\vert \mathbb {N} \vert$ Ôüá. This can be corrected by "skipping over" these numbers in the grid, using the Schr├ÂderÔÇôBernstein theorem, or by designing a function which does this naturally, for example using the CalkinÔÇôWilf tree.

A number is called algebraic if it is a solution of some polynomial equation with integer coefficients. For example, the square root of two Ôüá ${\sqrt {2}}$ Ôüá is a solution to Ôüá $x^{2}-2=0$ Ôüá, and the rational number Ôüá $p/q$ Ôüá is the solution to Ôüá $qx-p=0$ Ôüá. Conversely, a number which cannot be the root of any polynomial is called transcendental. Two examples include Euler's number (*e*) and pi (¤Ç). In general, proving a number is transcendental is considered to be very difficult, and only a few classes of transcendental numbers are known. However, it can be shown that the set of algebraic numbers is countable by ordering the polynomials lexicographically (for example, see *Cantor's first set theory article ┬º The proofs*). Since the set of algebraic numbers is countable while the real numbers are uncountable (shown in the following subsection), the transcendental numbers must form the vast majority of real numbers, even though they are individually much harder to identify. That is to say, almost all real numbers are transcendental.

#### Hilbert's hotel

Hilbert's paradox of the Grand Hotel is a popular thought experiment devised by the German mathematician David Hilbert to illustrate a counterintuitive property of countably infinite sets, allowing them to have the same cardinality as a proper subset of themselves. The scenario begins by imagining a hotel with an infinite number of rooms, one for each natural number, all of which are occupied. But then a new guest walks in asking for a room. The hotel accommodates by moving the occupant of room 1 to room 2, the occupant of room 2 to room 3, room 3 to room 4, and in general, room n to room n+1. Then every guest still has a room, but room 1 is open for the new guest.

Then, the scenario continues by imagining an infinitely long bus of new guests seeking a room. The hotel accommodates by moving the person in room 1 to room 2, room 2 to room 4, and in general, room n to room 2n. Thus, all the even-numbered rooms are occupied, but all the odd-numbered rooms are vacant, leaving room for the infinite bus of new guests. The scenario continues further by assuming an infinite number of these infinite buses arrive at the hotel, and showing that the hotel is still able to accommodate. Finally, an infinite bus which has a seat for every real number arrives, and the hotel is no longer able to accommodate.

### Uncountable sets

A set is called *uncountable* if it is not countable; that is, it is infinite and strictly larger than the set of natural numbers. The usual first example of this is the set of real numbers Ôüá $(\mathbb {R} )$ Ôüá, which can be understood as the set of all numbers on the number line. One method of proving that the reals are uncountable is called Cantor's diagonal argument, credited to Cantor for his 1891 proof, though his method differs from the more common presentation.

It begins by assuming, by contradiction, that there is some one-to-one mapping between the natural numbers and the set of real numbers between 0 and 1 (the interval Ôüá $[0,1]$ Ôüá). Then, take the decimal representation of each real number, for example, Ôüá $0.5772...$ Ôüá with a leading zero followed by any sequence of digits. The number 1 is included in this set since 1 = 0.999... Considering these real numbers in a list, it is always possible to create a new number such that the first digit of the new number is different from that of the first number in the list, the second digit is different from the second number in the list, and so on. The new number must also have a unique decimal representation, that is, it cannot end in repeating nines or repeating zeros. For example, if the digit is not 2, make the digit of the new number 2, and if it was 2, make it 3. Then, this new number will be different from each of the numbers in the list by at least one digit, and therefore must not be in the list. This shows that the real numbers cannot be put into a one-to-one correspondence with the naturals, and thus must be strictly larger.

Another classical example of an uncountable set, established using a related reasoning, is the power set of the natural numbers, denoted Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá. This is the set of all subsets of Ôüá $\mathbb {N}$ Ôüá, including the empty set and Ôüá $\mathbb {N}$ Ôüá itself. The method is much closer to Cantor's original diagonal argument. Again, assume by contradiction that there exists a one-to-one correspondence Ôüá f Ôüá between Ôüá $\mathbb {N}$ Ôüá and Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá, so that every subset of Ôüá $\mathbb {N}$ Ôüá is assigned to some natural number. These subsets are then placed in a list, in the order defined by Ôüá f Ôüá (see image). If Ôüá $1\in f(1)$ Ôüá, then Ôüá $1\notin T$ Ôüá, that is, if 1 is in the first subset of the list, then 1 is *not* in the subset Ôüá T Ôüá. Further, if Ôüá $2\notin f(2)$ Ôüá, then Ôüá $2\in T$ Ôüá, that is if the number 2 is not in the second subset of the list, then 2 *is* in the subset Ôüá T Ôüá. Then in general, for each natural number Ôüá n Ôüá, Ôüá $n\in T$ Ôüá if and only if Ôüá $n\notin f(n)$ Ôüá, meaning Ôüá n Ôüá is put in the subset Ôüá T Ôüá only if the nth subset in the list does not contain the number Ôüá n Ôüá. Then, for each natural number Ôüá n Ôüá, Ôüá $T\neq f(n)$ Ôüá, meaning, Ôüá T Ôüá is not the nth subset in the list, for any number Ôüá n Ôüá, and so it cannot appear anywhere in the list defined by Ôüá f Ôüá. Since Ôüá f Ôüá was chosen arbitrarily, this shows that every function from Ôüá $\mathbb {N}$ Ôüá to Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá must be missing at least one element, therefore no such bijection can exist, and so Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá must be not be countable.

These two sets, Ôüá $\mathbb {R}$ Ôüá and Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá can be shown to have the same cardinality (by, for example, assigning each subset to a decimal expansion) called the cardinality of the continuum.

Cantor's theorem generalizes the second theorem above, showing that every set is strictly smaller than its powerset. In broad strokes, the proof goes as follows: Given a set Ôüá A Ôüá, assume by contradiction that there is a bijection Ôüá f Ôüá from Ôüá A Ôüá to Ôüá ${\mathcal {P}}(A)$ Ôüá. Then, the subset Ôüá $T\subseteq A$ Ôüá given by taking the negation of the "diagonal", formally, Ôüá $T=\{a\in A:a\notin f(a)\}$ Ôüá, cannot be in the list. Therefore, every function is missing at least one element, and so Ôüá $\vert A\vert <\vert {\mathcal {P}}(A)\vert$ Ôüá. Further, since Ôüá ${\mathcal {P}}(A)$ Ôüá is itself a set, the argument can be repeated to show Ôüá $\vert A\vert <\vert {\mathcal {P}}(A)\vert <\vert {\mathcal {P}}({\mathcal {P}}(A))\vert$ Ôüá. Taking Ôüá $A=\mathbb {N}$ Ôüá, this shows that Ôüá ${\mathcal {P}}({\mathcal {P}}(\mathbb {N} ))$ Ôüá is even larger than Ôüá ${\mathcal {P}}(\mathbb {N} )$ Ôüá, which was already shown to be uncountable. Repeating this argument shows that there are infinitely many "sizes" of infinity.


## Cardinal numbers

In the above sections, "the cardinality of a set" was described relationally. In other words, one set could be compared to another, intuitively comparing their "size". Cardinal numbers are a means of measuring this "size" more explicitly. For finite sets, this is simply the natural number found by counting the elements. This number is called the *cardinal number* of that set, or simply *the cardinality* of that set. The cardinal number of a set Ôüá A Ôüá is generally denoted by Ôüá $\vert A\vert$ Ôüá, with a vertical bar on each side, though it may also be denoted by Ôüá A Ôüá, Ôüá $\operatorname {card} (A)$ Ôüá, or Ôüá $\#A$ Ôüá.

For infinite sets, "cardinal number" is somewhat more difficult to define formally. However, cardinal numbers are not usually thought of in terms of their formal definition, but immaterially in terms of their arithmetic/algebraic properties. The assumption that there is *some* cardinality function Ôüá $A\mapsto \vert A\vert$ Ôüá which satisfies Ôüá $A\sim B\iff \vert A\vert =\vert B\vert$ Ôüá (in prose, "two sets are equinumerous exactly when they map to the same cardinal number"), sometimes called the *axiom of cardinality* or *Hume's principle*, is sufficient for deriving most properties of cardinal numbers.

Commonly in mathematics, if a relation satisfies the properties of an equivalence relation, the objects used to materialize this relation are equivalence classes, each of which group objects equivalent to one another. These equivalence classes are called the *FregeÔÇôRussell* cardinal numbers. However, these equivalence classes are too large to form sets (apart from the cardinal number Ôüá ${\mathbf {0}}$ Ôüá whose only element is the empty set), since, for example, the cardinal number Ôüá ${\mathbf {1}}$ Ôüá would be the set of all sets with one element, then Ôüá $\{{\mathbf {1}}\}\in {\mathbf {1}}$ Ôüá, and would therefore contain itself, violating regularity. Thus, due to John von Neumann, it is more common to assign representatives of these classes.

### Finite cardinals

Given a basic sense of natural numbers, a set is said to have cardinality Ôüá n Ôüá if it can be put in one-to-one correspondence with the set Ôüá $\{1,\,2,\,\dots ,\,n\}$ Ôüá, analogous to counting its elements. For example, the set Ôüá $S=\{A,B,C,D\}$ Ôüá has a natural correspondence with the set Ôüá $\{1,2,3,4\}$ Ôüá, and therefore is said to have cardinality 4. Other terminologies include "Its cardinality is 4" or "Its cardinal number is 4". In formal contexts, the natural numbers can be understood as some construction of objects satisfying the Peano axiomsÔÇöa list of properties, such that any system satisfying these properties is, in a certain sense, just like the natural numbers.

Showing that such a correspondence exists is not always trivial. Combinatorics is the area of mathematics primarily concerned with counting, both as a means and as an end to obtaining results, and certain properties of finite structures. The notion cardinality of finite sets is closely tied to many basic combinatorial principles, and provides a set-theoretic foundation to recover them. It can be shown by induction on the possible sizes of sets that finite cardinality corresponds uniquely with natural numbers (cf. *Finite set ┬º Uniqueness of cardinality*). This is related to several other concepts, verifying Hume's principle and the basis of bijective proofs, and is equivalent to a certain formulation of the pigeonhole principle, that a finite set cannot be put in one to one correspondence with a proper subset of itself.

The addition principle asserts that given disjoint sets Ôüá A Ôüá and Ôüá B Ôüá, Ôüá $\vert A\cup B\vert =\vert A\vert +\vert B\vert$ Ôüá, intuitively meaning that the sum of the parts is equal to the whole. The multiplication principle asserts that given two sets Ôüá A Ôüá and Ôüá B Ôüá, Ôüá $\vert A\times B\vert =\vert A\vert \cdot \vert B\vert$ Ôüá, intuitively meaning that there are Ôüá $\vert A\vert \cdot \vert B\vert$ Ôüá ways to pair objects from these sets. Both of these can be proven by a bijective proof, together with induction. The more general result is the inclusionÔÇôexclusion principle, which defines how to count the number of elements in overlapping sets.

Naturally, a set is defined to be finite if it is empty or can be put in correspondence with the set Ôüá $\{1,\,2,\,\dots ,\,n\}$ Ôüá, for some natural number Ôüá n Ôüá. However, there exist other definitions of "finite" which do not rely on a definition of "number." For example, a set is called Dedekind-finite if it cannot be put in one-to-one correspondence with a proper subset of itself, though this definition requires the axiom of choice to be equivalent to the original definition.

### Aleph numbers

The aleph numbers are a sequence of cardinal numbers that represent the sizes of infinite sets, denoted with an aleph Ôüá $\aleph$ Ôüá, the first letter of the Hebrew alphabet. The first aleph number is Ôüá $\aleph _{0}$ Ôüá, called "aleph-nought", "aleph-zero", or "aleph-null", which represents the cardinality of the set of all natural numbers: Ôüá $\aleph _{0}=\vert \mathbb {N} \vert =\vert \{0,1,2,3,\cdots \}\vert$ Ôüá. Then, Ôüá $\aleph _{1}$ Ôüá represents the next largest cardinality, then Ôüá $\aleph _{2}$ Ôüá, and so on. The most common way this is formalized in set theory is through Von Neumann ordinals, known as Von Neumann cardinal assignment.

Ordinal numbers generalize the notion of *order* to infinite sets. For example, 2 comes after 1, denoted Ôüá $1<2$ Ôüá, and 3 comes after both, denoted Ôüá $1<2<3$ Ôüá. Then, one defines a new number, Ôüá $\omega$ Ôüá, which comes after every natural number, denoted Ôüá $1<2<3<\cdots <\omega$ Ôüá. Further Ôüá $\omega <\omega +1<\dots <\omega +\omega =\omega \cdot 2$ Ôüá, and so on. More formally, these ordinal numbers can be defined as follows:

Ôüá $0:=\{\}$ Ôüá, the empty set, Ôüá $1:=\{0\}$ Ôüá, Ôüá $2:=\{0,1\}$ Ôüá, Ôüá $3:=\{0,1,2\}$ Ôüá, and so on. Then one can define Ôüá $m<n{\text{, if }}\,m\in n$ Ôüá, for example, Ôüá $2\in \{0,1,2\}=3$ Ôüá, therefore Ôüá $2<3$ Ôüá. Defining Ôüá ${\displaystyle \omega$ Ôüá gives Ôüá $\omega$ Ôüá the desired property of being the smallest ordinal greater than all finite ordinal numbers. Further, Ôüá $\omega +1:=\{0,1,\cdots ,\omega \}$ Ôüá, and so on.

Since Ôüá $\omega \sim \mathbb {N}$ Ôüá by the natural correspondence, one may define Ôüá $\aleph _{0}$ Ôüá as the set of all finite ordinals. That is, Ôüá $\aleph _{0}:=\omega$ Ôüá. Then, Ôüá $\aleph _{1}$ Ôüá is the set of all countable ordinals (all ordinals Ôüá $\alpha$ Ôüá with cardinality Ôüá $\vert \alpha \vert \leq \aleph _{0}$ Ôüá), the first uncountable ordinal. Since a set cannot contain itself, Ôüá $\aleph _{1}$ Ôüá must have a strictly larger cardinality: Ôüá $\aleph _{0}<\aleph _{1}$ Ôüá. Furthermore, Ôüá $\aleph _{2}$ Ôüá is the set of all ordinals with cardinality less than or equal to Ôüá $\aleph _{1}$ Ôüá, and in general the successor cardinal Ôüá $\kappa ^{+}$ Ôüáis the set of all ordinals with cardinality up to Ôüá $\kappa$ Ôüá. Put another way for infinite cardinals, Ôüá $\kappa ^{+}$ Ôüá is the number of possible well-orderings on Ôüá $\kappa$ Ôüá up to order isomorphism. Proving that such a set always exists is known as Hartogs' theorem, wherein the smallest ordinal not less or equal to than a set Ôüá $\kappa$ Ôüá is called the Hartogs number of Ôüá $\kappa$ Ôüá. Then, Ôüá $\aleph _{\lambda }$ Ôüá for a limit ordinal Ôüá $\lambda$ Ôüá is the union of all lesser alephs.

The importance of ordinal numbers here is to generalize the notion of counting to infinite sets. When counting, one implicitly assigns an order to their set of objects, but no matter what order one assigns the final result of the count is always the same, which demonstrates the connection between cardinal and ordinal numbers. Further, by the well-ordering theorem, there cannot exist any set with cardinality between Ôüá $\aleph _{0}$ Ôüá and Ôüá $\aleph _{1}$ Ôüá, and every infinite set has some cardinality corresponding uniquely to some aleph Ôüá $\aleph _{\alpha }$ Ôüá, for some ordinal Ôüá $\alpha$ Ôüá. This allows one to use a constructive definition of the cardinality function, by assigning each set to its equinumerous aleph.

### Cardinal arithmetic

Basic arithmetic can be done on cardinal numbers in a very natural way, by extending the theorems for finite combinatorial principles above. The intuitive principle that if Ôüá A Ôüá and Ôüá B Ôüá are disjoint then addition of these sets is simply taking their union, written as Ôüá $\vert A\cup B\vert =\vert A\vert +\vert B\vert$ Ôüá. Thus if Ôüá A Ôüá and Ôüá B Ôüá are infinite, cardinal addition is defined as Ôüá ${\displaystyle \vert A\vert +\vert B\vert$ Ôüá where Ôüá $\sqcup$ Ôüá denotes disjoint union. Similarly, the multiplication of two sets is intuitively the number of ways to pair their elements (as in the multiplication principle), therefore cardinal multiplication is defined as Ôüá ${\displaystyle \vert A\vert \cdot \vert B\vert$ Ôüá, where Ôüá $\times$ Ôüá denotes the Cartesian product. These definitions can be shown to satisfy the basic properties of standard arithmetic:

- Associativity: Ôüá $\vert A\vert +\vert B\sqcup C\vert =\vert A\sqcup B\vert +\vert C\vert$ Ôüá, and Ôüá $\vert A\vert \cdot \vert B\times C\vert =\vert A\times B\vert \cdot \vert C\vert$ Ôüá
- Commutativity: Ôüá $\vert A\vert +\vert B\vert =\vert B\vert +\vert A\vert$ Ôüá, and Ôüá $\vert A\vert \cdot \vert B\vert =\vert B\vert \cdot \vert A\vert$ Ôüá
- Distributivity: Ôüá $\vert A\vert \cdot \vert B\sqcup C\vert =\vert A\times B\vert +\vert A\times C\vert$ Ôüá

Although a lot of properties of finite arithmetic hold for infinite arithmetic, as above and in the table below, strict inequalities (such as K┼ænig's theorem) are rare. For example, in finite arithmetic, for any nonzero number Ôüá n Ôüá, Ôüá $n+n>n$ Ôüá. However, since both the set of even numbers Ôüá $(E)$ Ôüá and set of odd numbers Ôüá $(F)$ Ôüá have cardinality Ôüá $\aleph _{0}$ Ôüá, it shows Ôüá $\vert E\sqcup F\vert =\vert \mathbb {N} \vert$ Ôüá thus Ôüá $\aleph _{0}+\aleph _{0}=\aleph _{0}$ Ôüá. In fact, for any infinite cardinal, Ôüá $\kappa +\kappa =\kappa \cdot \kappa =\kappa$ Ôüá. In this way, infinite cardinal addition and multiplication are considered to be remarkably well-behaved (at least under the Axiom of Choice).

A combinatorial argument can be used to show Ôüá $2^{\vert A\vert }=\vert {\mathcal {P}}(A)\vert$ Ôüá, by considering the indicator function of each subset. In general, cardinal exponentiation is not as well-behaved as addition and multiplication. For example, even though it can be proven that the expression Ôüá $2^{\aleph _{0}}$ Ôüádoes indeed correspond to some aleph, it is unprovable from standard set theories which aleph it corresponds to.

### Set of all cardinal numbers

The *set of all cardinal numbers* refers to a hypothetical set which contains all cardinal numbers. Such a set cannot exist, which has been considered paradoxical, and related to the Burali-Forti paradox. Using the definition of cardinal numbers as representatives of their cardinalities. It begins by assuming there is some set Ôüá $S:=\{X\,\vert X{\text{ is a cardinal number}}\}$ Ôüá. Then, if there is some largest cardinal Ôüá $\kappa \in S$ Ôüá, then the powerset Ôüá $2^{\kappa }$ Ôüá is strictly greater, and thus not in Ôüá S Ôüá. Conversely, if there is no largest element, then the union Ôüá $\bigcup S$ Ôüá contains the elements of all elements of Ôüá S Ôüá, and is therefore greater than or equal to each element. Since there is no largest element in Ôüá S Ôüá, for any element Ôüá $x\in S$ Ôüá, there is another element Ôüá $y\in S$ Ôüá such that Ôüá $\vert x\vert <\vert y\vert$ Ôüá and Ôüá $\vert y\vert \leq {\Bigl \vert }\bigcup S{\Bigr \vert }$ Ôüá. Thus, for any Ôüá $x\in S$ Ôüá, Ôüá $\vert x\vert <{\Bigl \vert }\bigcup S{\Bigr \vert }$ Ôüá, and so Ôüá ${\Bigl \vert }\bigcup S{\Bigr \vert }\notin S$ Ôüá. Thus, the collection of all cardinal numbers is too large to form a set, and is a proper class. Georg Cantor denoted the collection of all cardinal numbers **Î¬ÔÇÄ** (tav, the last letter of the Hebrew alphabet), and considered it to be an "inconsistent multiplicity".


## Cardinality of the continuum

The real numbers Ôüá $(\mathbb {R} )$ Ôüá formalize the intuitive notion of the *continuum*: the unbroken, gapless set of points on the number line. There are numerous ways to construct and analyze the continuum in set theory, for example, sets of Cauchy sequences of rational numbers, or Dedekind cuts. However, a somewhat informal definition as the set of infinite sequences of digits, e.g. Ôüá $0.1234...$ Ôüá, with the usual arithmetic and order is a popular and sufficient choice for basic arguments involving the continuum. The cardinality of this set, denoted "Ôüá ${\mathfrak {c}}$ Ôüá" (a lowercase fraktur script "c"), turns out to be remarkably stable under various transformations.

For example, all intervals on the real line e.g. Ôüá $[0,1]$ Ôüá, and Ôüá $[0,2]$ Ôüá, have the same cardinality as the entire set Ôüá $\mathbb {R}$ Ôüá. First, Ôüá $f(x)=2x$ Ôüá is a bijection from Ôüá $[0,1]$ Ôüá to Ôüá $[0,2]$ Ôüá. The fact that two intervals of unequal length can be mapped one-to-one has been known, even if considered paradoxical, for thousands of years (cf. *┬º Ancient history*). Further, the tangent function is a bijection from the interval Ôüá $\textstyle \left({\frac {-\pi }{2}}\,,{\frac {\pi }{2}}\right)$ Ôüá to the whole real line. A more surprising example is the Cantor set, which is defined as follows: take the interval Ôüá $[0,1]$ Ôüá and remove the middle third Ôüá $\textstyle ({\frac {1}{3}},{\frac {2}{3}})$ Ôüá, then remove the middle third of each of the two remaining segments, and continue removing middle thirds (see image). The Cantor set is the collection of points that survive this process. The remaining points are exactly those whose decimal expansion can be written in ternary without a 1. Reinterpreting these decimal expansions as binary (e.g. by replacing the 2s with 1s) gives a bijection between the Cantor set and the interval Ôüá $[0,1]$ Ôüá. The Cantor set has a measure or "length" of zero, and yet is still equinumerous to the whole real line.

In the other direction, into higher dimensions, one can find a bijective map between a one-dimensional line and a two-dimensional square. Given a point in the unit square in Ôüá $\mathbb {R} ^{2}$ Ôüá (two-dimensional space) Ôüá $\textstyle (0.a_{1}a_{2}a_{3}...\,,0.b_{1}b_{2}b_{3}...)$ Ôüá, one can interleave their digits to get the unique number Ôüá $\textstyle 0.a_{1}b_{1}a_{2}b_{2}...$ Ôüá in the unit interval Ôüá $[0,1]$ Ôüá. Space-filling curves offer a more visual proof, giving a continuous map from the unit interval onto the unit square. Classical examples include the Peano curve and Hilbert curve. Although these maps are not bijective, they are indeed sufficient to show Ôüá $\vert \mathbb {R} ^{2}\vert \leq \vert \mathbb {R} \vert$ Ôüá with the converse being immediate. Both of these methods can be reused at each dimension to show that Ôüá $\vert \mathbb {R} ^{n}\vert =\vert \mathbb {R} ^{n+1}\vert$ Ôüá and thus Ôüá $\vert \mathbb {R} \vert =\vert \mathbb {R} ^{n}\vert$ Ôüá for any dimension Ôüá $n\geq 1$ Ôüá. Further, the infinite cartesian product Ôüá $\mathbb {R} ^{\infty }$ Ôüá can also be shown to be equinumerous to Ôüá $\mathbb {R}$ Ôüá by cardinal arithmetic: Ôüá $\textstyle \vert \mathbb {R} ^{\infty }\vert =\vert \mathbb {R} \vert ^{\aleph _{0}}=(2^{\aleph _{0}})^{\aleph _{0}}=2^{(\aleph _{0}\cdot \aleph _{0})}=2^{\aleph _{0}}=\vert \mathbb {R} \vert$ Ôüá. Thus, the real numbers, all finite-dimensional real spaces, and the countable cartesian product have the same cardinality.

As shown in ┬º Uncountable sets, the set of real numbers is strictly larger than the set of natural numbers. Specifically, Ôüá $\vert \mathbb {R} \vert =\vert {\mathcal {P}}(\mathbb {N} )\vert$ Ôüá. The Continuum Hypothesis (CH) asserts that the real numbers have the next largest cardinality after the natural numbers, that is Ôüá $\vert \mathbb {R} \vert =\aleph _{1}$ Ôüá. As shown by G├Âdel and Cohen, the continuum hypothesis is independent of ZFC, a standard axiomatization of set theory; that is, it is impossible to prove the continuum hypothesis or its negation from ZFCÔÇöprovided that ZFC is consistent, meaning it produces no contradictions. The Generalized Continuum Hypothesis (GCH) extends this to all infinite cardinals, stating that Ôüá $2^{\aleph _{\alpha }}=\aleph _{\alpha +1}$ Ôüá for every ordinal Ôüá $\alpha$ Ôüá. Research on CH and GCH continues independent of ZFC, especially in descriptive set theory and through the exploration of large cardinal axioms. Without GCH, the cardinality of Ôüá $\mathbb {R}$ Ôüá cannot be written in terms of specific alephs. The Beth numbers ( Ôüá $\textstyle \beth$ Ôüá, the second letter of the Hebrew alphabet) provide a concise notation for powersets of the real numbers starting from Ôüá $\beth _{0}=\vert \mathbb {N} \vert$ Ôüá, then Ôüá $\beth _{1}=2^{\beth _{0}}=\vert \mathbb {R} \vert$ Ôüá, and Ôüá $\beth _{2}=\vert {\mathcal {P}}(\mathbb {R} )\vert =2^{\beth _{1}}$ Ôüá, and in general Ôüá $\beth _{n+1}=2^{\beth _{n}}$ Ôüá and Ôüá $\beth _{\lambda }=\bigcup _{\alpha <\lambda }\beth _{\alpha }$ Ôüá if Ôüá $\lambda$ Ôüá is a limit ordinal.
